**handleProxyRequest(req: Request, res: Response): Promise<void>**

Implementation flow:
1. Parse model from request body: `const modelName = req.body?.model || 'claude-opus-4-20250514'`
2. Resolve route: `const resolution = providerService.resolveModelRoute(modelName)` — if null, emit Anthropic error: "No route configured for model: {modelName}"
3. Get API key: `const apiKey = await keychainService.getKey(resolution.provider.name)` — if null, emit error: "API key not found for provider: {name}"
4. Select adapter: Use `getOrCreateAdapter(resolution.provider.providerType || resolution.provider.name, resolution.provider.baseUrl)` — if the provider name doesn't match a known type, fall back to CustomAdapter
5. Transform request: `const providerBody = adapter.transformRequest(req.body, resolution)`
6. Make upstream request with timeout (per D-21):
   - Create AbortController
   - Set timeout: `adapter.timeouts.streaming` (120s)
   - POST to `${resolution.provider.baseUrl}/v1/messages` with headers: Content-Type: application/json, Authorization: Bearer ${apiKey}, Accept: text/event-stream
   - Body: JSON.stringify(providerBody)
   - Signal: controller.signal
7. Set response headers: Content-Type: text/event-stream, Cache-Control: no-cache, Connection: keep-alive, X-Accel-Buffering: no
8. Transform and stream response: `for await (const event of adapter.transformResponse(upstreamResponse, { messageId: \`msg_${crypto.randomUUID()}\`, model: req.body.model, inputTokens: 0 })) { res.write(event); }`
9. res.end()
10. Catch errors: clearTimeout, emit Anthropic error via emitAnthropicError(res, error)

**emitAnthropicError(res: Response, error: unknown): void**
- Set SSE headers
- Use getUserFacingErrorMessage() to get sanitized message (per D-28)
- Log full error internally with console.error (without API keys)
- Write error SSE event: `event: error\ndata: ${JSON.stringify({type: 'error', error: {type: 'api_error', message: userFriendlyMessage}})}\n\n`
- res.end()

Import: Request, Response from 'express'; getAdapter/getOrCreateAdapter from './adapters/index.js'; providerService from './services/provider.js'; keychainService from './services/keychain.js'; getUserFacingErrorMessage, formatSSEEvent from './services/sse-transformer.js'.

Remove the old createProxyMiddleware import and createProxyHandler function entirely.

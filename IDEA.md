#Claude code proxy
voglio creare un app in next.js o react per mac che farà da proxy ad claude code
deve poter avere piu router:
- openrouter
- opencode zen/go
- ollama
- custom

quello che voglio fare è un app che mi permetta per ogni modello di claude code di mappare un modello da un altro provider 
es 
opus --> opencode/qwen3.6
sonnet --> openrouter/mimo-v2-flash
haiku --> opencode/nvidia/nemotron-3-super-120b-a12b:free

in reference ci sono 2 repo di github belline che ho trovato che funziono bene dalla quale potremmo prendere ispirazione o direttamente copiarla e modificarla (claude-code-router e free-claude-code)
ci sono poi due miei progetti (reference/) che non funzionano ma che ti possono dare l'idea di come voglio l'app. non prenderli come spunto ma guarda solo l'UX dell'app.

per il design c'è un file DESIGN.md che puoi consultare.

studia le reference che ti ho dato, in particolar modo "claude-code-router" e "free-claude-code" (specialmente la seconda) poi crea l'app.
